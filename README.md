# EDA-of-sales-data

Краткое описание проекта: Исследование потребительского поведения в четырех городах (Москва, СПб, Новосибирск, Екатеринбург) с целью оптимизации рекламного бюджета и сегментации аудитории.

## Технологический стек

* **Язык**: Python
* **Библиотеки**: Pandas, Matplotlib, Seaborn
* **Статистические методы**: Анализ распределений, работа с ненормально распределенными данными, анализ сводной таблицы, проведение однофакторного дисперсионного анализа, проверка гипотез.
